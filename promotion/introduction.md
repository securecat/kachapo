"Kachapo" is a Chrome extension that visualizes the sounds of your interactions on any webpage. It was inspired by "Miruoto," an ethical creative project by Hogaku inc. (founded by Reiko Katayama). That said, unlike Miruoto, Kachapo solves absolutely no social problems — it's just a joke app.

While browsing, your clicks, drag-selections, and keystrokes in input fields are all accompanied by onomatopoeia that pops up on screen.

Note that Kachapo may not work on sites with strict CSP (Content Security Policy) or heavy SPA frameworks. Consider it best-effort fun.

Source code is available on GitHub: https://github.com/securecat/kachapo

「カチャポ」はウェブページ上での操作音を可視化するChrome拡張です。株式会社方角（代表：方山れいこ氏）によるエシカルクリエイティブ「ミルオト」に触発されて開発しました。ただし、カチャポはミルオトとは異なり、何の社会課題も解決しない単なるジョークソフトです。

ウェブページ閲覧中のクリック、テキスト選択時のドラッグ、入力欄でのキータイプといった操作に、擬音が視覚的に付きます。

SPAやCSP（コンテンツセキュリティポリシー）が厳しいサイトでは動作しないことがありますので、ベストエフォートでお楽しみください。

ソースコードはGitHubで公開しています：https://github.com/securecat/kachapo
